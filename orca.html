Orca Explorer

An open-source Roblox GUI tool for developers to inspect and interact with in-game events.

Orca Explorer is an open-source project designed for Roblox developers. It provides a sleek, modern interface to explore and manipulate in-game events like RemoteEvent, RemoteFunction, BindableEvent, and BindableFunction. Check out the source code on GitHub.
